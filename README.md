# vue-draggable-div

## 说明
vue可拖拽可伸缩的组件，目前只是借鉴，详见以下  传送门

## 安装
```
npm install vue-draggable-div
```

## 使用
```
<script>
import VueDraggableDiv from 'vue-draggable-div'
    ...
components: {VueDraggableDiv}
```

## 打包
打包使用命令 npm run build-bundle
